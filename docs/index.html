[04] The secret sauce

Why retrieval, not fine-tuning.

For compliance content, grounding beats fine-tuning — it can cite its source, and it never goes stale.

✗ Fine-tuning the model

  • Facts baked into weights — can't cite a source
  • Goes stale the moment a policy changes
  • Still hallucinates; no refusal on unknowns
  • Retraining cost on every document change

✓ Retrieval-augmented (ours)

  • Every answer traceable to an exact passage
  • Updates instantly when a document is uploaded
  • Refuses when the answer isn't in the documents
  • No retraining — swap the docs, done
